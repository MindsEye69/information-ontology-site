# Information Ontology Project

## Quick Reference

**CodeGraph commands for this project:**
```powershell
codegraph sync              # Update index after code changes (do this at session start)
codegraph status            # Check index health
codegraph query <symbol>    # Find a symbol (e.g., "LegendComponent")
codegraph context "<task>"  # Get relevant code for a task
codegraph impact <symbol>   # What breaks if you change this?
codegraph callers <symbol>  # Where is this called from?
codegraph callees <symbol>  # What does this call?
```
